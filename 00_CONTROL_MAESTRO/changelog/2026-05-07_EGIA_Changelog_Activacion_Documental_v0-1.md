# Changelog

## v0.1 - Activación documental inicial
**Fecha:** 2026-05-07  
**Estado:** Activo  
**Responsable documental:** Ricardo Juan Morales De Jesús, Ph.D.

### Cambios principales

1. Se confirmó el nombre del proyecto: **Educación General e Inteligencia Artificial en Acción**.
2. Se definió la Facultad de Estudios Generales como base institucional inicial.
3. Se definió la comunidad universitaria general como público amplio, con énfasis en docentes e investigadores.
4. Se estableció el proyecto como ruta permanente.
5. Se definió el horizonte **Full N4**.
6. Se confirmó el nivel operativo inicial como **N3 avanzado**.
7. Se estableció el ecosistema técnico-documental:
   - Local;
   - VS Code;
   - GitHub;
   - Obsidian;
   - WordPress futuro.
8. Se conectó el cartapacio local con VS Code y GitHub.
9. Se creó una bóveda de Obsidian asociada al cartapacio local.
10. Se decidió comenzar por fundamentos teóricos antes de métodos, protocolos y plantillas.
11. Se seleccionó como primer documento conceptual el puente:
    `De calendario semestral a ecosistema formativo evolutivo`.
12. Se generaron archivos de continuidad:
    - Compendio de Situación;
    - Primer de Seguimiento;
    - Prompt de Activación para Nueva Conversación.
13. Se generaron archivos operativos de cierre:
    - Bitácora de Sesión;
    - Changelog;
    - Registro de Decisiones.
14. Se recomendó crear una matriz de fuentes antes de iniciar revisión bibliográfica extensa.
15. Se definió el próximo documento:
    `Fundamentos Teóricos de Educación General, IA Ética y Tecnologías Emergentes`.
