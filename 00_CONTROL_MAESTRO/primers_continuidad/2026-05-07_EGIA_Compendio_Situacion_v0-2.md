# Compendio de Situación
## Educación General e Inteligencia Artificial en Acción

## 1. Estado general del proyecto

El proyecto **Educación General e Inteligencia Artificial en Acción** queda activado como ruta permanente de formación, investigación, documentación activa, colaboración y transferencia universitaria sobre tecnologías emergentes e inteligencia artificial ética y responsable.

El proyecto nace a partir de la inspiración de **Viernes de IA**, pero ya no se limita a un calendario semestral. Viernes de IA queda preservado como antecedente, semilla metodológica, memoria histórica y prototipo formativo inicial.

La transición conceptual acordada es:

```text
Calendario semestral
→ ruta formativa
→ laboratorio documentado
→ comunidad de práctica
→ proyectos transversales
→ ecosistema universitario permanente
```

## 2. Repositorio y ecosistema técnico

El cartapacio local ya fue conectado con:

- **VS Code**, para edición, control local y trabajo con archivos;
- **GitHub**, como repositorio versionado;
- **Obsidian**, como bóveda o segundo cerebro asociada al cartapacio local.

El ecosistema previsto del proyecto es:

```text
Local / VS Code
  → producción, edición y control inicial

GitHub
  → fuente versionada, issues, releases, manifest, control de cambios

Obsidian
  → segundo cerebro, navegación conceptual, enlaces, mapas de conocimiento

WordPress
  → capa pública futura para recursos curados, publicaciones y comunicación
```

## 10. Línea futura: EGIA Quest

Se propone desarrollar una aplicación educativa lúdica, tentativamente llamada **EGIA Quest**, para motivar a participantes mediante rutas por niveles, retos, puntos, badges, dilemas éticos, prácticas de accesibilidad, mini-bitácoras, competencias múltiples y exportación de perfil y progreso en JSON portable.
