# Guía de Exportación MD / JSON v0.1
## EGIA-Accion

## Principio

Markdown será el formato humano primario. JSON será el formato computable para manifest, evidencia, decisiones, metadatos y automatización futura.

## Reglas

1. Todo documento humano estable debe existir en `.md`.
2. Todo registro computable debe poder representarse en `.json`.
3. Los archivos exportables deben incluir fecha, tipo, versión y estado.
4. No exportar datos sensibles sin anonimización.
5. No publicar evidencias de participantes sin autorización.

## Nomenclatura

```text
AAAA-MM-DD_EGIA_TipoDocumento_Tema_v#-#.md
AAAA-MM-DD_EGIA_TipoDocumento_Tema_v#-#.json
```
