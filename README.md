# EGIA Quest 🧭✨  
## Portafolio lúdico, ético y portable para aprender IA en acción

**EGIA Quest** es un prototipo educativo diseñado para acompañar el proyecto **Educación General e Inteligencia Artificial en Acción**. Su propósito es ayudar a participantes universitarios a practicar destrezas con inteligencia artificial, documentar su progreso, reflexionar éticamente, atender la accesibilidad y construir un portafolio exportable en formatos abiertos.

Esta versión inicial está pensada para comenzar de forma sencilla:

```text
HTML + JSON + Markdown
sin servidor
sin autenticación
sin GitHub Actions
listo para GitHub Pages
```

EGIA Quest no es un juego de puntos vacíos. Es una brújula formativa para aprender, crear, revisar, cuidar y transferir evidencias de trabajo con IA. 🧠🌱

---

## 🌟 Visión general

EGIA Quest propone una experiencia lúdica universitaria donde cada participante puede avanzar mediante retos, registrar reflexiones, resolver dilemas éticos, aplicar buenas prácticas de accesibilidad y exportar su progreso como portafolio personal.

La ruta se organiza alrededor de una idea central:

```text
Pregunta educativa
→ práctica situada
→ producto
→ evidencia
→ reflexión ética
→ documentación activa
→ archivo
→ transferencia
→ continuidad
```

Cada acción deja huella.  
Cada reto produce evidencia.  
Cada evidencia puede convertirse en aprendizaje, memoria, investigación o transferencia, siempre con consentimiento y privacidad por defecto.

---

## 🔐 Privacidad por defecto

EGIA Quest parte de una regla clara:

> el perfil pertenece al participante.

Por defecto:

- se usa alias
- no se solicita correo
- no se solicita identificación institucional
- no se incluyen evidencias privadas
- no se publica nada automáticamente
- no se envía nada a terceros
- la exportación se puede revisar antes de descargar
- el perfil local se puede eliminar

Las evidencias pueden clasificarse como:

| Clasificación | Uso |
|---|---|
| Privada | Solo para la persona participante |
| Archivable | Puede guardarse con consentimiento |
| Investigable | Puede analizarse con consentimiento específico |
| Publicable | Puede compartirse públicamente con autorización explícita |

## 🚀 Cómo usar esta versión

### Opción local

1. Descarga los archivos.
2. Abre `index.html` en tu navegador.
3. Crea tu perfil con alias.
4. Explora retos.
5. Registra mini-bitácoras.
6. Exporta tu progreso.

No necesitas instalar nada.

---

### Opción GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube `index.html`, este `README.md` y los documentos de apoyo.
3. Entra a:

```text
Settings → Pages
```

4. Selecciona:

```text
Deploy from a branch
main / root
```

5. Guarda.

Tu prototipo quedará disponible en una dirección parecida a:

```text
https://TU-USUARIO.github.io/egia-quest/
```
