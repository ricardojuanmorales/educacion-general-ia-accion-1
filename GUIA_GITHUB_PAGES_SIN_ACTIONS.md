# Guía rápida: publicar EGIA Quest v0.1A en GitHub Pages sin Actions

## 1. Crear repositorio

Crea un repositorio en GitHub:

```text
egia-quest
```

## 2. Subir archivos

Sube estos archivos a la raíz:

```text
index.html
README.md
EGIA_Quest_MVP_HTML_JSON_MD_v0_1A.md
MODELO_JSON_EGIA_QUEST_v0_1A.md
GUIA_GITHUB_PAGES_SIN_ACTIONS.md
```

## 3. Activar Pages

En GitHub:

```text
Settings → Pages → Build and deployment
```

Selecciona:

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

Guarda.

## 4. Abrir URL

GitHub mostrará una URL parecida a:

```text
https://TU-USUARIO.github.io/egia-quest/
```

## 5. Reglas antes de publicar

- Usa solo datos simulados.
- No subas perfiles reales.
- No subas evidencias privadas.
- No subas bitácoras de participantes sin consentimiento.
- No publiques exportaciones JSON personales.
