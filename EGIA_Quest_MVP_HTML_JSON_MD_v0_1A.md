# EGIA Quest v0.1A
## MVP monolítico HTML + JSON + Markdown

## Decisión técnica

Esta versión simplifica el MVP original React/TypeScript y lo convierte en un prototipo estático publicable directamente en GitHub Pages.

## Objetivo

Probar la experiencia central antes de invertir en arquitectura más compleja:

```text
Perfil local
→ retos
→ dilemas éticos
→ mini-bitácora
→ badges
→ privacidad por defecto
→ exportación JSON/Markdown
```

## Principio

Primero validar flujo pedagógico. Luego escalar tecnología.

## Componentes incluidos

1. Pantalla Mi Perfil y Progreso.
2. Catálogo de retos.
3. Dilemas éticos.
4. Mini-bitácora.
5. Exportador JSON.
6. Exportador Markdown.
7. Reglas de privacidad por defecto.
8. Documentación para GitHub Pages.

## No incluye

- Autenticación.
- Servidor.
- Base de datos remota.
- GitHub Actions.
- React.
- npm.
- Analítica.
- Publicación automática de evidencias.

## Regla ética

Nada se envía automáticamente a terceros. El participante revisa, copia, descarga o elimina su perfil local.
