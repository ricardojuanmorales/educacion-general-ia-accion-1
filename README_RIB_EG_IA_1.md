**Estado:** Activación documental inicial v0.1

## Propósito

Este repositorio organiza la primera versión local del proyecto **Educación General e Inteligencia Artificial en Acción**, una ruta permanente de formación, investigación, documentación activa, acompañamiento y transferencia sobre inteligencia artificial en Educación General.

El proyecto nace inspirado en **Viernes de IA**, pero evoluciona hacia una arquitectura permanente con bitácora, evidencia robusta, gobernanza documental, archivo vivo, estudio independiente guiado y publicación progresiva.

## Fórmula rectora

```text
Pregunta educativa → práctica situada → producto → evidencia → reflexión ética → documentación activa → archivo → transferencia → continuidad
```

## Cómo usar este repositorio

1. Abrir este cartapacio en **VS Code**.
2. Revisar `00_CONTROL_MAESTRO/` primero.
3. Usar `manifest_egia_accion.json` como mapa computable del proyecto.
4. Conectar el repositorio con **GitHub** para versionado.
5. Abrir la estructura también como vault de **Obsidian** para navegación conceptual.
6. Publicar solo materiales curados y autorizados en **WordPress**.
